Запуск программы (Windows):

!Перед запуском файлов .bat в них необходимо поменять пути.

Для запуска необходимо выполнить файл run.bat (если переменные среды не установлены, то необходимо вызвать startup.bat). 
После запуска в браузере откроется страница с формой, где необходимо ввести скаляр в формате HEX-строки и нажать кнопку "multiply". 
Далее начнется умножение скаляра на точку эллиптической кривой с использованием webasm, после завершения операции перемножения выведутся координаты полученной точки и потраченной время (12с).
Для компиляции кода C в webasm необходимо выполнить файл compile_main.bat.
Стандартные параметры ЭК хранятся в файле main.cpp. Также их можно задать при вызове функции в js multiply вместо multiply_with_default (вызывается по умолчаниию).
